Code implementing the TCI analyses described in this paper:

Norman-Haignere SV, Long LK, Devinsky O, Doyle W, Irobunda I, Merricks EM, Feldstein NA, Schevon CA, Flinker A, Mesgarani N. Multiscale temporal integration organizes hierarchical computation in human auditory cortex. Nature Human Behavior.

example_data.m illustrates how to apply the code to the example electrodes from the paper.
